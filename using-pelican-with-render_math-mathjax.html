Using Pelican with render_math (mathjax)

Date Sun 15 January 2017 Tags pelican / publishing

Finally got Pelican to work with render_math. The key is to modify the math.py from your render_math submodule directory with changes according to this.

Afterwards, just add latex equations to the markdown file, and then can render equation like:

$$e=mc^2$$
